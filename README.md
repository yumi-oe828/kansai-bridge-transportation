# kansai-bridge-transportation

📌 概要
関西ブリッジ運送のWebサイトのために制作した、WordPressオリジナルテーマです。

🛠 使用技術
WordPress
PHP
HTML / CSS
JavaScript

💡 実装内容
front-page.phpによるトップページ構築
template-partsによるパーツ分割
カスタムテーマとして構築

🚀 工夫した点
再利用しやすい構造設計（header / footer / parts）
ACFやCPTUIなど適切なプラグインを使用し構築
レスポンシブ対応

📁 ディレクトリ構成
kansai-bridge/
├── front-page.php
├── functions.php
├── template-parts/
├── assets/
├── style.css

⚠️ 注意
このリポジトリはWordPressテーマのみを含んでいます。
WordPress本体は含まれていません。
