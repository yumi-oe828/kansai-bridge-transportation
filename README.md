# kansai-bridge-transportation

📌 概要

関西の橋と交通情報を紹介するWebサイトのために制作した、WordPressオリジナルテーマです。

🛠 使用技術
WordPress
PHP
HTML / CSS
JavaScript
💡 実装内容
front-page.phpによるトップページ構築
template-partsによるパーツ分割
カスタムテーマとして構築
🚀 工夫した点
再利用しやすい構造設計（header / footer / parts）
視認性を意識したUI設計
レスポンシブ対応
📁 ディレクトリ構成

kansai-bridge/
├── front-page.php
├── functions.php
├── template-parts/
├── assets/

⚠️ 注意

このリポジトリはWordPressテーマのみを含んでいます。
WordPress本体は含まれていません。
